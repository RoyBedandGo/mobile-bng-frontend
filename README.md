# BedandGo System Mobile (React Native)

The official mobile application frontend for the BedandGo real estate "Super App" and property marketplace. Built with React Native and Expo, this application provides a fast, secure, and seamless interface that connects directly to the BedandGo cloud backend.

## 🚀 Tech Stack

- **Framework:** React Native (via Expo)
- **API Client:** Axios (configured with global request interceptors)
- **Authentication:** Expo SecureStore (for secure JWT management)
- **Environment:** Node.js

## ⚙️ Getting Started

### Prerequisites

Make sure you have Node.js installed, along with the Expo CLI. If you are testing on a physical device, download the **Expo Go** app on your iOS or Android phone.

### 1. Installation

Clone this repository and install the required dependencies:

```bash
npm install
# or
yarn install
```
